# Contributing

Thank you for your interest in contributing to Forgata's Proximity Workstation Lock.

## How to contribute

1. Fork the repository.
2. Create a feature branch.
3. Open a pull request describing your changes.

## Reporting issues

Please open issues for:

- bugs or incorrect behavior
- documentation improvements
- new hardware or platform compatibility

## Code style

- TypeScript should compile with `npm run build`.
- Python should follow a consistent style and remain readable.

## Pull requests

Provide a clear summary of changes, the motivation, and any testing steps.
If the change affects configuration or startup, include updated README guidance.
